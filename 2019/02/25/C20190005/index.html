认知为什么是重要的

发表于 2019-02-25 | 分类于 职业力

  认知为什么是重要的？

  技术的弱势是一时的，认知的弱才是真的弱。技术可以理解为完成一件事情的方法，而认知就是决策能力，规划能力，视野，格局。

  在中国所有的企业家中，马云的格局可谓是最大的。在互联网萌芽之际，马云只因为在美国敲了几下键盘就认识到这背后的巨大可能性。首先，马云是比绝大多数人先接触到了互联网，也就是认知上快人一步。而且，认知不仅仅是先后之分，还有思考的深度。我们有理由相信，当时接触到互联网的绝不只有马云，可是在思考的，并思考得非常深的，绝对包括马云。

  对事物的深度思考，能提高我们的认知水平。任正非有一次开会对员工说：”知道我为什么比你们水平高吗？”，众人皆不知，他答道：“因为我从每件事情中学到的比你们多”。时间上我们每个人只有24小时，我们每天遇到的事情数目上大体差不多。所以我们要抽出时间思考，思考能让很多东西产生质变。

  至此，我们总结认知的两个关键点：快速了解新动向，对事物深入思考。

  几年前，大家还喜欢拿京东和阿里巴巴比较，阿里巴巴的营收涨了，jd的股票就跌。然而，前者早就跳开了淘宝这个局，阿里巴巴依靠阿里云和蚂蚁金服在新的领域再造了一个又一个淘宝，而淘宝也一分为三，向不同方向努力。马云扶贫基金会2018年的报告振奋人心，它宣告阿里人在承担大企责任，主动去做改变人类，改变世界的行动。遗憾的是，jd现在只能和天猫拼多多做比较。虽然我不认同宿命论，但是认知的差距早就决定了故事的结局。

  在大学校园中，很多大学生把日子过得恍惚，或自我娱乐而玩世不恭，或按部就班随大流走。现下最严重的现象莫过于等到大三大四才开始有紧迫感，并程序化地考研考公或培训。这种人是被推着走的，一旦施加在你身上的力撤了，如断线风筝。现实中，笔者被很多的师弟师妹问到如何应对招聘，该如何选择。很多时候我是抱着无语的心态并撑着知心学长的角色在回答。我很难理解为什么到了大三下学期，还只是开始学习某种基础知识，还不知道要如何提升自己，还不能自我学习。我发现大多数的提问者对本专业对应的行业毫无认知可言，把糟糕的教科书奉为圭臬，对死板的老教师唯命是从，学着完全无法投入生产的甚至不能称为技术的不知道什么东西。我思考了很久，发现根源其实就是认知。他们对这个行业的认知太少，井底之蛙罢了。

  优秀的大学生，从大一开始就应该主动去了解社会需求，从学长那里了解绝对是前期最有效的途径。还应该去关注行业动态，行业龙头企业，行业的相关技术论坛等。要主动加入一些相关的群体，增加认知场景。一旦你了解到企业的需求，你就不会只把目光放在游戏或者教科书中了，我们将从整个行业的角度去看待自己。认知的场景其实现在很多，微信公众号，学习类app，学习类的小团体，社区，都是很棒的途径。当然，只有场景是不够的，要加深理解，要主动去思考每个新闻背后的含义，把自己代入去思考“为什么这件事情他们这么做了”，“如果是你又该如何”。要不断提高认知的深度。

  第一次听到认知这个概念是在傅盛的文章里，那时候感触不深，后面结合很多的思考，发现其实很多成功最开始都是认知上的领先。所以我从来不鼓励盲目地做选择，被迫地做选择。因为盲目意味着认知的缺乏，如前言，认知的差距早就决定了结局。世界上没有什么选择是绝对正确的，但是盲目选择大概率就是错误的，错就错在认知。
